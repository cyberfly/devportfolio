# Plum - Project Management Tools

## Background

Our team hates Jira and want to replace with something else. We then build Plum and end up replacing Jira.

## Features

- Todoist inside task
- Global discussion stream
- Markdown editor
- Mention

## Stack

- VueJS
- TailwindCSS
- Django with Django Rest API
