# Tailwind Layout

TailwindLayout is a collection of popular layouts and patterns made with Tailwind CSS.

At LaLoka Labs, I propose the idea and build this site to serve as our Front End reference and we decided to open source it.

## Stack

- NuxtJs 2 (VueJs)
- TailwindCSS
